# Portfolio-2.0

Portfolio Website – HTML & CSS Assignment

Overview
This project is a responsive single-page portfolio website built using HTML, CSS, and JavaScript. The website presents a developer profile, projects, and a contact form while demonstrating basic frontend development concepts such as layout design, responsiveness, and simple interactivity.

Implementation

1. HTML Structure

The website is organized into semantic sections:
Header / Navigation – Contains the website logo, navigation links (Home, Hobbies, About, Projects, Contact), and a Resume button.
Hero Section – Displays the main introduction with a title, short description, and call-to-action buttons.
Hobbies Section – Shows a hobby description and a list of benefits.
About Section – Brief introduction and statistics about experience and completed projects.
Projects Section – Displays selected work using project cards.
Contact Section – Includes contact details and a form for user messages.
Footer – Contains copyright information and social links.

2. CSS Styling

CSS is used to:
Create a dark themed layout similar to the provided UI.
Structure the layout using Flexbox and CSS Grid.
Style components such as buttons, cards, forms, and navigation.
Implement responsive design using media queries so the website works on both desktop and mobile screens.
